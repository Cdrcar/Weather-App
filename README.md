# Weather Dashboard ☀️
Server-Side APIs

A weather dashboaard that allows the user to search for a city and obtain the current weather for that specific city as well as a 5-day weather forecast ☁️

## Built with:
- HTML
- CSS
- JavaScript
- Jquery
- Libraries
- API
